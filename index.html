Instalacion de software

En esta pagina explicaremos la instalacion de distintos tipos de software. Nos centraremos en estos 4 tipos y tomaremos un ejemplo de cada uno en donde explicaremos que son y los pasos para instalarlo en tu ordenador. Estos son los que utilizaremos como ejemplo:

Software ofimático

En primer lugar, esta el software ofimatico del cual explicaremos que es, daremos un ejemplo de una aplicación gratuita y explicaremos su instalación, además de hacer una comparativa con otro software ofimático de pago.

Software de recuperación

Por otro lado haremos una introducción de que es un software de recuperación y tambien daremos los pasos necesarios para instalar este una aplicación gratuita.

Software de seguridad

Tambien, instalaremos un antivirus el cual lo compararemos con otro de su mismo tipo para valorar si merecera la pena instalarlo o no.

Software portable

Por ultimo, daremos un ejemplo de software portable para poder hacer un pendrive de arranque y diferenciaremos la versión portable y la no portable de esta aplicación.
