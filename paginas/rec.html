Software de recuperación:

Este tipo de software permite al usuario recuperar sus datos en situaciones críticas, tales como fallos en el disco duro, fallos del sistema, fallos en el sistema operativo, archivos dañados o eliminación accidental de archivos de una serie de dispositivos de almacenamiento y últimos gadgets, como SSD, cámaras de vídeo, unidad flash, etc.

Para ello usaremos Hiren's BootCD que básicamente lo que hace es que mediante un disco que se denomina como "de arranque", cuenta con una serie de utilidades para poder mejorar el rendimiento del equipo y en caso de que tenga fallas, pueda realizar las modificaciones correspondientes.

Instalación

Para su instalación, en primer lugar, iremos a la pagina principal de este software.

Ya dentro de la pagina, nos iremos a la ventana de descargas y bajaremos a la tabla en la que se puede instalar el archivo iso

Con eso ya instalado crearemos un pendrive de arranque con el archivo iso ya instalado.
