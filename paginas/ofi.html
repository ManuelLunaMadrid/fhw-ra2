Despues de haber entrado en la pagina oficial, accederemos a la pestaña de descarga que se encuentra en la parte superior de la pagina, y aparecerá lo siguiente:

Hay varias versiones de libre office, pero he elegido la version 7.2.2. Encima del boton de descarga, esta la opción de elegir el sistema operativo y al elegirlo descargo el paquete.

Y por ultimo, al abrir el paquete, aparecera el asistende de instalacion de libre office que te ayudara a instalarlo o modificarlo como desees.

Comparacion de requisitos de LibreOffice y Microsof Office 365

Los prerrequisitos de hardware y software para instalar LibreOffice son los siguientes:

  • Microsoft Windows 7 SP1, Windows 8, Windows Server 2012, o Windows 10
  • PC Pentium compatible (Pentium III, Athlon, aunque se recomienda un sistema más reciente)
  • 256 MB de RAM (se recomiendan 512 MB de RAM)

Por otro lado, para Microsoft Office 365 los requisitos de hardware mínimos son los siguientes:

  • Procesador de 1 GHz o más rápido con el conjunto de instrucciones de SSE2.
  • 1 GB o más de memoria RAM para 32 bits, 2 GB o más de memoria RAM para 64 bits.
  • Espacio disponible en disco duro de 3 GB o más.

En el caso de Microsoft Office 365 tenemos que proporciona mayor comodidad en el uso y mas opciones de edicion de los archivos, aunque este no es un software gratis ni libre.

En cambio, en el caso de LibreOffice, este es un software libre gratuito que te permite realizar la gran mayoria de las opciones de Microsoft por lo que este puede ser una mejor opcion.
